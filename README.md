# 🍕 Pizza Order Dashboard – Nextjs + Firebase Google Auth

A sleek **Pizza Order Dashboard** built using **Next** with Ts and **Firebase Authentication**. This app allows users to **log in with Google** and view a **pizza ordering interface**. It's perfect for showcasing login flows, dashboard UI, and authentication handling.

---

## 🚀 Features

- 🔐 Google Login with Firebase Authentication
- 👤 Display user profile (name, email, photo)
- 🍕 Dynamic Pizza Order Dashboard UI
  - View pizza varieties Stats
  - Real-time UI updates
- 🧭 Protected routes (only accessible after login)
- 🧾 Responsive layout using modern React components
- 🌐 Deployed live on Vercel

---

## 🧑‍💻 Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/XSTRANGER-7/PiePiZZ
cd your-repo
```
### 2. Install dependencies
```
npm i
```
### 3. Generate Your Firebase Config & copy in .env file
```bash
VITE_API_KEY=your_api_key
VITE_AUTH_DOMAIN=your_project.firebaseapp.com
VITE_PROJECT_ID=your_project_id
VITE_STORAGE_BUCKET=your_project.appspot.com
VITE_MESSAGING_SENDER_ID=your_sender_id
VITE_APP_ID=your_app_id
```
### 4. Run Your Code
```bash
npm run dev
```
